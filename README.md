## ğŸš€ Getting Started

### 1. Clone and install

```bash
git clone https://github.com/ChrisR08/react-game-app.git
cd react-game-app
npm install
```

### 2. Set up your environment variables

Create a `.env.local` file in the root:

```env
VITE_API_BASE_URL=https://api.rawg.io/api
VITE_API_KEY=your_rawg_api_key_here
```

### 3. Run locally

```bash
npm run dev
```
